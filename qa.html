Question-1: What are the different ways to select an element in the DOM?

getElementsByTagName() , getElementsByClassName() , getElementById() , querySelector() , querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML reads both the HTML markup and the text content of the element innerText read the content of an element textContent returns or sets all text including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where an event starts from the target element and propagates up to its parents, triggering event listeners on parent elements unless stopped with event.
